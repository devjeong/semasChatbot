# 소진공 AI 챗봇 플러그인

## 개요

소진공 AI 챗봇은 IntelliJ IDEA IDE에 내장되어 개발 생산성을 향상시키는 AI 비서 플러그인입니다. LM Studio와 같은 로컬 LLM(거대 언어 모델) 서버와 연동하여 동작하므로, 소스 코드가 외부로 유출될 걱정 없이 안전하게 사용할 수 있습니다.

## 주요 기능

### 1. 지능형 코드 분석 및 질의응답
- **코드 선택 후 질문:** 에디터에서 분석하고 싶은 코드 조각을 선택하고, `Ctrl+Shift+C` 단축키 또는 마우스 우클릭 메뉴의 `Send Selection to Chat`을 실행하세요.
- **챗봇과 대화:** 챗봇 패널이 활성화되며 선택한 코드 정보가 표시됩니다. 이제 해당 코드에 대해 궁금한 점을 질문하거나, 코드와 관련 없는 일반적인 대화를 나눌 수 있습니다.

### 2. AI 기반 코드 수정 제안 및 적용
- **자연어 기반 코드 수정:** "이 함수에 주석 추가해줘" 또는 "이 코드 개선해줘" 와 같이 자연어로 코드 수정을 요청할 수 있습니다.
- **인라인(In-line) 수정 제안:** AI의 수정 제안은 에디터 내에서 직접 표시됩니다.
  - 원본 코드는 노란색으로 하이라이트 처리됩니다.
  - 하이라이트된 코드 바로 아래에 **[✅ 적용]** / **[❌ 거절]** 버튼이 나타납니다.
- **원클릭 적용 및 거절:**
  - **✅ 적용:** 제안을 수락하면, 원본 코드가 AI가 수정한 코드로 즉시 교체됩니다.
  - **❌ 거절:** 제안을 거절하면, 하이라이트와 버튼이 사라지고 원본 코드는 그대로 유지됩니다.
- **안전하고 직관적인 리팩토링:** 여러 군데를 수정해야 하는 복잡한 요청도 AI가 자동으로 처리해주므로, 안전하고 정교한 코드 관리가 가능합니다.

### 3. 로컬 LLM 연동
- 모든 AI 기능은 사용자가 지정한 로컬 LLM 서버(예: LM Studio)를 통해 처리됩니다.
- 소스 코드가 외부 네트워크로 전송되지 않아, 보안이 중요한 프로젝트에서도 안심하고 사용할 수 있습니다.

## 시작하기

1. 플러그인을 설치하고 IDE를 재시작합니다.
2. 우측 툴바에서 "SemasChatbot" 아이콘을 클릭하여 챗봇 패널을 엽니다.
3. `LmStudioClient.kt` 파일에서 로컬 LLM 서버 주소를 확인하고, 필요하다면 수정합니다. (기본값: `http://localhost:1234/v1/chat/completions`)
4. 이제 AI 챗봇과 함께 더 빠르고 효율적으로 코딩 작업을 시작해보세요!
